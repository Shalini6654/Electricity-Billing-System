# Electricity-Billing-System
This is a GUI made using Java Swing and awt. It lets user perform multiple operations like:-
1. Add new customer.
2. Generate monthly bills.
3. View customer details.
4. User can pay electricity bill.
5. User can create his/her personal login for security purpose.

ABOUT THIS PROJECT:- 
This Java application is created using Visual Studio Code. Additional library is added for the support of JDBC(JDBC is using to connect a Java based application with a database to store the data of user). This project contains 11 different classes which are:-
1. SplashScreen.class
2. Login.class
3. AddC.class
4. AddC2.class
5. Main.class
6. CalculateBill.class
7. PayBill.class
8. Conn.class(A class file connecting java application with MySQL)
9. Customer_Details.class
10. generate_bill.class
11. LastBill.class

DATABASE (MySQL)
Database for this Electricity Billing System contains 4 tables :-
1. Login(username, password)
2. addc(name, meter_no, address, state, city, email, phone_no)
3. bill(month, meter_no, units_consumed, amount)
4. tax(id, meter_location, meter_type, phase_code, bill_type, days, meter_rent, mcb_rent, service_rent, gst)
